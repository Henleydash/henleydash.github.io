Leçon 1 : Introduction au Langage C

Par : Hassan Dao

1. Qu'est-ce que le C ?

Le C est un langage compilé, performant et très utilisé dans les systèmes embarqués et les systèmes d'exploitation.

2. Structure d'un programme

Voici votre premier programme qui affiche "Bonjour UTS" :

/* Mon premier programme en C */
#include <stdio.h>

int main() {
    printf("Bonjour UTS !\n");
    return 0;
}

3. Les étapes pour exécuter ce code

  • Écriture : Utiliser un éditeur (comme Coding C).
  • Compilation : Transformer le texte en binaire avec gcc.
  • Exécution : Lancer le fichier obtenu.
