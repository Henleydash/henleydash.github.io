Python : Niveau Débutant 🐍

Module 1 : Introduction et premier script

1. Pourquoi Python à l'UTS ?

Python est un langage interprété. Contrairement au C, vous n'avez pas besoin de gérer manuellement la mémoire. Sa syntaxe est simple, ce qui permet de se concentrer sur la logique algorithmique.

💡 Info : Python est massivement utilisé à l'université pour l'analyse de données et l'intelligence artificielle.

2. Votre premier programme

En Python, pas besoin de #include ou de main() pour commencer. Une seule ligne suffit :

# Ceci est un commentaire en Python
print("Bonjour l'Université Thomas Sankara !")

# Calcul simple
resultat = 10 + 5
print("Le résultat est :", resultat)

3. Comment pratiquer sur Android ?

Pour coder en Python directement sur votre smartphone, je vous recommande d'installer l'application Pydroid 3 disponible sur le Play Store. Elle inclut tout le nécessaire pour exécuter vos scripts sans ordinateur.
